案例19：编写一个批量添加用户脚本
在 server0 上创建 /root/batchusers 脚本
1）此脚本要求提供用户名列表文件作为参数
2）如果没有提供参数，此脚本应该错误提示 Usage: /root/batchusers，退出并返回相应值
3）如果提供一个不存在的文件，此脚本应该给出错误提示 Input file not found，退出并返回相应值
4）新用户的登录Shell为 /bin/false，无需设置密码
5）用户列表测试文件：http://classroom.example.com/pub/materials/userlist 

# vim userlist.txt
natasha
chihiro
harry
sarah
# vim /root/batchusers
if [ $# -eq 0 ];then
  echo 'Usage: /root/bachusers' >&2
  exit 2
elif [ -f $1 ];then
  for i in `cat $1`
  do
  useradd -s /bin/false $1
  echo "$i创建成功"
  done
else
  echo 'Input file not found' >&2
  exit 3
fi
